Shopping Cart Page

A dynamic shopping cart web application built using HTML, CSS, and JavaScript.

##Features

- Display cart items
- Increase and decrease quantity
- Remove products from cart
- Auto-update total bill
- Product images
- Clean user interface

##Technologies Used

- HTML
- CSS
- JavaScript

##How to Run

1. Download the project
2. Open "index.html" in browser
